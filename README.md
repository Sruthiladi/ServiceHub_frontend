# ServiceHub

ServiceHub is a web application that connects users with skilled professionals. Users can browse services, search by category, book professionals, and manage their bookings, while professionals can manage their services and requests through a dedicated dashboard.

## Features

* User, Professional, Admin and Support roles
* User authentication
* Browse and search for services
* Category-based filtering
* Book professionals
* Professional dashboard
* Responsive user interface

## Tech Stack

* React
* JavaScript
* HTML
* CSS
